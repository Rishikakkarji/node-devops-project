# Node DevOps Docker Project

This project containerizes a Node.js application using Docker Compose with MongoDB and Redis.

## Technologies

Node.js  
Docker  
Docker Compose  
MongoDB  
Redis  

## Run Project 
## IF you are using window machine then please install docker desktop.
docker compose up --build
## If you are using linux then install the docker container along with compose package.
docker compose up --build

## Application URL

http://localhost:8090

## Health Check

http://localhost:8090/health
